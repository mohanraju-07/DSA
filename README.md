# Project: Data Structures and Algorithms (DSA) 

## Project Overview
===============

The Data Structures and Algorithms (DSA) project is an extensive collection of C implementations for various data structures and algorithms. The project is divided into nine folders, each containing specific data structures and algorithms.

## Folder Structure & Explanation
=================================

### .\divide and conquer

*   This folder contains implementations of various divide-and-conquer algorithms in C, including binary search, merge sort, matrix multiplication, and finding the minimum and maximum value in an array.

### .\graph

*   This folder contains C implementations for various graph algorithms and data structures, including detection of cycles, Depth-First Search (DFS), Breadth-First Search (BFS), and Prim's algorithm for finding the minimum spanning tree.

### .\greedy approach

*   This folder contains implementations of various algorithms that utilize the Greedy Approach, including Dijkstra's algorithm for finding the shortest path in a graph and job scheduling using a greedy approach.

### .\Linked List

*   This folder contains implementations of three different types of linked lists in C, including singly, double, and circular linked lists.

### .\queue

*   This folder contains a simple implementation of a queue data structure in C, following the First-In-First-Out (FIFO) principle.

### .\Search Algorithms

*   This folder contains a C implementation of a linear search algorithm, a simple and naive method for finding an element in an unsorted array.

### .\stack

*   This folder contains several C programming language files that implement various stack-based data structures and algorithms, including balanced parentheses, infix-to-prefix conversion, and postfix evaluation.

### .\trees

*   This folder contains C implementations of various tree data structures, including AVL trees, binary search trees, and complete binary trees.

### .\hashing

*   This folder contains C implementations of various hash table data structures and algorithms, including basic hash table creation and search operations.

## Features
======

*   The project contains C implementations for various data structures, including linked lists, stacks, queues, and trees.
*   It also includes implementations of various algorithms, such as binary search, merge sort, Dijkstra's algorithm, and Prim's algorithm.
*   The project provides an extensive collection of example use cases and explanations for each data structure and algorithm.

## Technologies Used
=====================

*   Programming language: C
*   Development Environment: GCC compiler
*   Operating System: Linux

## How to Run the Project
=========================

1.  Clone the repository from GitHub.
2.  Navigate to the project directory.
3.  Compile the C files using the GCC compiler (e.g., `gcc -o binarysearch binarysearch.c`).
4.  Run the compiled executable file to test the data structures and algorithms.
